<span data-ttu-id="e085c-175">在 Program.cs 中執行 Main 方法，如下所示。</span><span class="sxs-lookup"><span data-stu-id="e085c-175">Implement the Main method in Program.cs as shown below.</span></span> <span data-ttu-id="e085c-176">此程式碼會建立內容的新實例，然後使用它來插入新的 Blog。</span><span class="sxs-lookup"><span data-stu-id="e085c-176">This code creates a new instance of our context and then uses it to insert a new Blog.</span></span> <span data-ttu-id="e085c-177">然後，它會使用 LINQ 查詢來抓取資料庫中依標題字母順序排序的所有 Blog。</span><span class="sxs-lookup"><span data-stu-id="e085c-177">Then it uses a LINQ query to retrieve all Blogs from the database ordered alphabetically by Title.</span></span>

``` csharp
class Program
{
    static void Main(string[] args)
    {
        using (var db = new BloggingContext())
        {
            // Create and save a new Blog
            Console.Write("Enter a name for a new Blog: ");
            var name = Console.ReadLine();

            var blog = new Blog { Name = name };
            db.Blogs.Add(blog);
            db.SaveChanges();

            // Display all Blogs from the database
            var query = from b in db.Blogs
                        orderby b.Name
                        select b;

            Console.WriteLine("All blogs in the database:");
            foreach (var item in query)
            {
                Console.WriteLine(item.Name);
            }

            Console.WriteLine("Press any key to exit...");
            Console.ReadKey();
        }
    }
}
```
