* <span data-ttu-id="7f05d-111">如果停用可為 null 的參考型別（預設值），則所有具有 .NET 參考型別的屬性都會依照慣例設定為選擇性（例如 `string`）。</span><span class="sxs-lookup"><span data-stu-id="7f05d-111">If nullable reference types are disabled (the default), all properties with .NET reference types are configured as optional by convention (e.g. `string`).</span></span>
