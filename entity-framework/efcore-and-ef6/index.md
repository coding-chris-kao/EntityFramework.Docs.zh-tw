## <a name="ef-core"></a><span data-ttu-id="33445-104">EF Core</span><span class="sxs-lookup"><span data-stu-id="33445-104">EF Core</span></span>

<span data-ttu-id="33445-105">Entity Framework Core ([EF Core](xref:core/index)) 是適用於 .NET 的新式物件資料庫對應程式。</span><span class="sxs-lookup"><span data-stu-id="33445-105">Entity Framework Core ([EF Core](xref:core/index)) is a modern object-database mapper for .NET.</span></span> <span data-ttu-id="33445-106">其支援 LINQ 查詢、變更追蹤、更新以及結構描述移轉。</span><span class="sxs-lookup"><span data-stu-id="33445-106">It supports LINQ queries, change tracking, updates, and schema migrations.</span></span>
